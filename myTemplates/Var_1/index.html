Задача #1


Какое из чисел большее:

||| ДАНО |||

--- 51 / 68
--- 17 / 34
--- 15 / 20
--- 111 / 333

||| РЕШЕНИЕ |||

Вариант ответа №1

111/333 < 51/68 < 17/34 < 15/20

Вывод: 15/20

Вариант ответа №2

-1) 51:68=0,75
-2) 17:34=0,5
-3) 15:20=0,75
-4) 111:333=0,333...

111:333 < 17:34 < 51:68 = 15:20

Вывод: оба варианта совокупно равны: 51:68 = 15:20 = 0,75


Задача #2


Сократить дробь 107100 / 119700. Ответ записать в виде несократимой дроби Х / У.

||| ДАНО |||

--- 107100 / 119700

||| РЕШЕНИЕ |||

107100 / 119700 =
(107100/119700):100 = (1071 / 1197):7 = (153 / 171):9 = 17 / 19
(сначала сократили на 100, затем на 7, затем на 9)

Вывод: 17 / 19


Задача #3


Предположим, что PIN-код состоит из двух цифр и латинской буквы. На то, чтобы ввести код один раз, уходит 1 секунда. Сколько времени потребуется, чтобы подобрать правильный PIN-код?

||| РЕШЕНИЕ |||

На комбинацию из двух чисел возможно не более 100 вариаций (то есть "99" + "0"). В латинском алфавите 26 букв (знаков). Таким образом получаем: 26(1 знак) умножить на 100(2 цифры) = 2.600 комбинаций (условных секунд). Максимальное время подбора PIN-кода составит 2.600 : 60 = 43,333...

Вывод: 43 минуты и 333... секунды
